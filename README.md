# Leer Nederlands 🐱

A Duolingo-style Dutch learning app with cute cats, built with React + Vite.

## Features
- 10 chapters from Start.nl coursebook (302 words)
- 7 exercise types: multiple choice, type, drag & match, fill-in-blank, listening, emoji, sentences
- 4 native languages: English, Farsi, Turkish, Arabic
- Spaced repetition for weak words
- Pronunciation via Web Speech API
- Gamification: XP, streaks, hearts, achievements
- Exam mode with timer
- Leaderboard
- Dark mode
- Mobile-optimized

## Development
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
```
Then deploy the `dist` folder to Vercel, Netlify, or any static host.
